# 🌍 Global Food Price Dashboard

An interactive Streamlit dashboard analysing global food prices across 55 countries and 482 commodities.

## Dataset
HDX Global Food Price Dataset (2026) — sourced from the Humanitarian Data Exchange (humdata.org)

## Features
- Price trends over time
- Country comparison
- Commodity breakdown
- Interactive world map
- Sidebar filters by country, category and commodity
